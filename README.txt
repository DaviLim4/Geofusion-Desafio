Para o desafio de Web Scraping, foi utilizado a linguagem Java com Selenium WebDriver.

Passos para execução:
	
	Para executar o script de coleta, será necessario abrir o prompt de comandos:
	1 - Teclas Win + R (se for sistema operacional windows);
	2 - Digitar "cmd" e acessar o prompt;
	3 - No prompt de comandos digitar "java -java LocalDoArquivo.Jar"
		Exemplo: "javba -jar C:\Users\DaviLima\IdeaProjects\geofusionDesafioScript\out\artifacts\geofusionDesafioScript_jar\geofusionDesafioScript.jar"
	OBS: O arquivo jar pode ser encontrado no diretório "geofusionDesafioScript\out\artifacts\geofusionDesafioScript_jar\geofusionDesafioScript.jar" do projeto.
	
	4 - Após executar o.jar o script será iniciado. O script irá pedir o lugar e nome do arquivo para que seja salvo.
		Exemplo:"C:/Users/DaviLima/Desktop/teste.csv"
		OBS: É importante colocar a extensão do arquivo (no caso .csv).
	5 - O script inicia uma execução automatica na WEB para coletar as informções;
	6 - Após o termino da execução, o script irá informar que o arquivo foi gerado com sucesso no diretório informado no passo "4".
